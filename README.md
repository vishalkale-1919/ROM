# Wedding Invitation Frontend Website

A modern responsive wedding invitation website built using:

- HTML5
- CSS3
- JavaScript ES6
- GSAP Animations
- AOS Scroll Animations
- Swiper.js Gallery
- Font Awesome

---

## Features

### Landing Section

- Full Screen Hero Banner
- Couple Names
- Wedding Date
- Call To Action Button

### Countdown Timer

- Days
- Hours
- Minutes
- Seconds

### Couple Section

- Bride Profile
- Groom Profile
- Animated Cards

### Story Timeline

- First Meeting
- Friendship
- Engagement
- Wedding

### Wedding Events

- Mehendi
- Sangeet
- Wedding
- Reception

### Gallery

- Swiper Slider
- Responsive Images

### Venue

- Embedded Google Maps

### RSVP

- Form Validation
- Submission Alert

### Animations

- GSAP ScrollTrigger
- Floating Flowers
- Fade In
- Slide In
- Zoom Effects
- Parallax Background

---

## Installation

Clone project:

```bash
git clone <repository-url>
```

Open project:

```bash
cd wedding-invitation-frontend
```

Run locally:

```bash
python -m http.server 8000
```

Visit:

```text
http://localhost:8000
```
